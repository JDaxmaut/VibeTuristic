Руководство администратора

Краткая инструкция по работе с сайтом Vibe Turistic в Wagtail CMS.

Туры

Основное

Туры — это страницы Wagtail. Открой Мои туры → нужный тур → Редактировать.

Вкладки редактора тура:

Основное

Название, бейдж, описание, длительность, цена, размер группы, время сбора.

Фото

Hero-фото, карточка для главной, фото для блоков «Что входит» и «Что не входит». Только JPEG, мин. 1200×800 px.

Программа

Маршрут по дням: заголовок, описание каждого дня.

Расписание

Заезды этого тура: даты, кол-во мест, статус. Добавленные здесь даты сразу видны на главной и странице тура.

Совет: чтобы тур появился на сайте, нажми «Опубликовать» внизу редактора. «Сохранить черновик» — изменения не видны посетителям.

Расписание заездов

Часто
  1. 1

    Открой тур: Мои туры → тур → Редактировать

  2. 2

    Перейди на вкладку Расписание

  3. 3

    Нажми «Добавить заезд» — укажи дату начала, дату конца, кол-во мест и статус

  4. 4

    Нажми «Опубликовать» — заезд мгновенно появляется на сайте

Снипеты

Отзывы / Контент

Отзывы

Имя, город, рейтинг 1–5, текст отзыва, цвет аватара (HEX). Все отзывы выводятся в разделе «Отзывы» на главной.

Почему мы

Иконка, цвет темы, заголовок, текст. Карточки блока «Почему Vibe Turistic». Порядок — поле «Порядок».

Заезды

Глобальный список всех заездов. Удобен для массового обновления статусов перед сезоном.

Заявки

Заявки с формы бронирования. Имя, контакт, тур, заезд, сообщение.
